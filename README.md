## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/slshotts77/MonthlyBudget.MVC
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
